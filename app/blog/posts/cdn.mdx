**Prompt for Visualization:** Without a CDN, every image request travels to your origin. If your server is in Virginia and the user is in India, 
that adds 250–300 ms of latency per request. With a CDN, the same image is served from a nearby edge server in 20–40 ms. 
That is a massive performance difference.
